### 🗺️ 双指缩放防护

**功能：**
- ✅ 防止用户双指缩放时触发浏览器缩放
- ✅ 双指操作仅影响地图，不影响页面
- ✅ 单指滑动、缩放仍正常工作

**实现方式：**
```javascript
document.querySelector("#map").addEventListener("touchmove", event => {
  if (event.touches.length > 1) {
    event.preventDefault();
  }
}, { passive: false });
```
